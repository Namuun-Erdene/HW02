Хичээлийн тодорхойлолт

Вэб технологийн талаарх суурь ойлголтуудыг олгож эдгээр технологиуд болон вэб хөгжүүлэхэд ашиглагддаг хэлнүүдийг ашиглан вэб хуудсыг хэрхэн хөгжүүлэхийг суралцах болно. Сүүлийн үеийн нэмэлт технологиуд болох вэб хөгжүүлэлтийн фраймворкуудын талаарх үндсэн ойлголтуудыг олгох сургалтын үйл ажиллагаа.

Хичээлийн зорилго

Вэб технологийн талаарх суурь ойлголтуудыг олгож өгөгдсөн вэб дизайны дагуу HTML5, CSS3, JS болон фраймворкийг ашиглан
мэргэжлийн програмуудыг тусламжтайгаар вэб хөгжүүлэлт хийх суурь мэдлэг, чадварыг олгох

Агуулга

CLO1: HTML5-н технологийг танин мэдэх, хөгжүүлэлт хийх, вэбийг хэрэглэгчдэд хэрхэн харуулах талаархи дадлагажуулах

Агуулга

CLO2: Вэб дизайн, үүлэн технологи (cloud service) талаар тайлбарлах, CLO3: CSS3-н технологийг танин мэдэх хөгжүүлэлт хийхэд дадлагажсан байх, CLO4: Програмчлалын хэл болон JavaScript хэлний ойлгох, хэрэглэх, CLO5: JavaScript ашиглан интерактив вэбсайт хөгжүүлэлт хийхэд дадлагажуулах
